# Copper Canister Corrosion Model

## Project Goal

This project models the long-term corrosion behaviour of a copper overpack used for spent nuclear fuel disposal in a deep geological repository.

## Objectives

- Temperature evolution
- Groundwater chemistry
- Copper corrosion kinetics
- Steel corrosion after copper failure
- Sensitivity analysis
- Lifetime prediction

## Programming Language

Python 3.10+

## Libraries

- NumPy
- Pandas
- Matplotlib
- SciPy
